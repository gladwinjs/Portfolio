Web Scraping Instagram Data

Project Description

This project involves scraping data from Instagram to gather insights about user profiles. The script collects data such as:

  • Username
  • Bio Description
  • Total number of followers and followees
  • List of followers
  • Number of posts
  • Number of likes and comments on each post

The data is then processed and visualized to provide valuable insights for social media analysis.

Iris Flower Classification

Project Description

The Iris Flower Classification project involves analyzing the famous Iris dataset, which contains measurements of three different iris species:

  • Iris Setosa
  • Iris Versicolor
  • Iris Virginica

Using this dataset, we plotted classification details and developed a model to predict the species of an iris flower based on its features. The project includes data visualization, model training, and evaluation.
